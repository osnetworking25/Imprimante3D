# Creality Ender 3 (2018) — carte 8 bits

## Contexte

Mon Ender 3 d'origine (2018), modèle de base (pas la version "Pro"), avec sa carte mère 8 bits d'origine.

## État actuel

Aucun travail n'a encore été fait sur cette imprimante dans ce projet — pas de firmware personnalisé, pas de BLTouch installé pour l'instant.

## À compléter plus tard

- Version exacte de la carte mère (souvent une carte "Melzi" ou similaire sur les tout premiers modèles).
- Si un BLTouch est prévu, refaire le même travail que pour l'Ender 3 Pro et la Hornet : sources Marlin correspondantes, activation du BLTouch, mesure des offsets, etc.
- Sur une carte 8 bits, attention à la mémoire flash très limitée : certaines options de confort (comme celles ajoutées sur l'Ender 3 Pro/Hornet) peuvent ne pas tenir en mémoire, il faudra être plus sélectif.
